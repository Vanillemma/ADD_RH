# Analyse statistique de données RH / Statistical Analysis of HR Data

## Français

Ce dépôt contient des scripts R réalisés pour aider à analyser des données RH fournies dans le cadre d’un projet d’étude.  
L’objectif principal était de produire des statistiques descriptives, nettoyer les données et faciliter l’interprétation des résultats.

## Contenu du dépôt

- `Analyse de donnees.R` :script principal lié à l’analyse des données RH.

## Objectifs du projet

Ce travail avait pour but de :

- importer et préparer les données ;
- nettoyer certaines variables si nécessaire ;
- produire des statistiques descriptives ;
- analyser les tendances principales dans les données RH ;
- aider à mieux comprendre les résultats à partir de scripts R.

## Confidentialité

Les données originales ne sont pas incluses dans ce dépôt afin de respecter la confidentialité des informations RH.

# Statistical Analysis of HR Data

## English

This repository contains R scripts created to help analyze HR data provided as part of a study project.  
The main goal was to produce descriptive statistics, clean the data, and support the interpretation of the results.

## Repository content

- `Analyse de donnees.R`: main script related to the HR data analysis.

## Project objectives

The purpose of this work was to:

- import and prepare the data;
- clean some variables when necessary;
- produce descriptive statistics;
- analyze the main trends in the HR data;
- support a better understanding of the results using R scripts.

## Confidentiality

The original data is not included in this repository in order to respect the confidentiality of HR information.  
The scripts are shared only to show the analysis process carried out with R.
